# Builder Tools

Static HTML/CSS/JavaScript frontend for building and renovation tools.

## Current Setup

- `index.html` is the main page.
- `assets/css/main.css` contains site styles.
- `assets/js/main.js` contains shared JavaScript.
- `src/pages` contains temporary static pages for future tools.
- `src/tools` is reserved for tool-specific logic.

Open `index.html` in a browser to preview the frontend.
